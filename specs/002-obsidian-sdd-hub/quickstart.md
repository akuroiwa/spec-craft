# Quickstart: Storyboard Integration

## Analyzing a Storyboard
1. Place a file in `obsidian/my_project.md`.
2. Ask the AI: "Analyze my project storyboard `my_project.md`."
3. The AI will use `read_storyboard` to understand the strategy.

## Getting Domain Help
1. Ask the AI: "Help me design a character for my manga."
2. The AI will reference the `manga-storyboard-guide` prompt.

## Checking Extensions
1. Ask the AI: "Can I generate a video right now?"
2. The AI will use `check_agent_extensions` to see if video tools are installed.
