# Quickstart: Phase 3 Features

## Automated Tactics
1. Ask the AI: "Run the pre-execution checks for me."
2. The AI will use `execute_tactical_script` to run `check-prerequisites.sh`.

## Setting up a New Project
1. Start a new spec-kit project.
2. Ask the AI: "Bootstrap my constitution with spec-craft rules."
3. The AI will use `bootstrap_constitution` to provide the necessary Markdown blocks.

## Agent Help
1. Ask the AI: "How do I add an extension to Claude Code?"
2. The AI will use `agent_extension_guide` to point you to `.claude/commands/`.
