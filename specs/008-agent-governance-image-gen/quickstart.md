# Quickstart: Generative Workflows

## Finding Extensions
1. Ask the AI: "Where can I find more extensions for Gemini CLI?"
2. The AI will point you to `https://geminicli.com/extensions/`.

## Checking Capabilities
1. Ask the AI: "Can I generate a manga scene now?"
2. The AI uses `check_generative_capabilities` and either confirms or provides an install command.

## Generating Assets
1. Define a scene in `obsidian/ja/scene1.md`.
2. Tell the AI: "Generate the illustration for scene 1 based on the storyboard."
3. The AI follows the `generative-workflow-guide` to execute the build.
