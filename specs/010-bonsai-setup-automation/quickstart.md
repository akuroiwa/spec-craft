# Quickstart: Bonsai & Setup

## Checking Your Environment
1. Run:
   ```bash
   spec-craft check-env
   ```
   Or ask the AI: "Check if my SDD environment is ready."
2. Follow the suggested installation commands for any missing tools.

## Architectural Design (Bonsai)
1. In Obsidian, define your building elements (e.g., walls, floors).
2. Ask the AI: "Generate the Bonsai script for my architectural strategy."
3. The AI will use the `bonsai-architectural-guide` to create the Python script.

## Releasing to Test PyPI
1. Follow the `test-pypi-release-checklist` prompt to build and upload your package safely.
